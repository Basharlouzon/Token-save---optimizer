## Platform Format Differences

### Kilo Format (`.kilo/skills/*/SKILL.md`)
- No frontmatter required
- Standard markdown with clear sections
- Reference project-relative paths

### Claude Code Format (`.claude/skills/*/SKILL.md`)
- YAML frontmatter: `---\nname: ...\ndescription: ...\n---`
- Same content body as Kilo (can be identical)

### Antigravity Format (root `SKILL.md`)
- YAML frontmatter: `---\nname: ...\ndescription: ...\nrisk: low\nsource: local\n---`
- The root SKILL.md should remain as the enhanced tokenso-optimizer (it's the primary skill)
